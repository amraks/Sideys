README for Peg Solitaire	
	
This is a computer program for triangular Peg Solitaire with 15 pegs. You can 
read about the game rules at http://home.comcast.net/~gibell/pegsolitaire/tindex.html. 
	
This program can solve the puzzle for any board configuration which is possible. 
If you want to see the program in action visit http://harshsavla.info/PegGame/peggame.html
